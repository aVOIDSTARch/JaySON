    Class JsonMaker

    Main class for JSON Schema validation and JSON file operations. Provides a unified interface for loading schemas, validating data, reading/writing JSON files, transforming data, and generating code.

    JsonMaker

    Example

    const maker = new JsonMaker("./schemas");

    // Validate data
    const result = maker.validate({ name: "John" }, "user.json");

    // Generate TypeScript types
    const tsCode = maker.generateTypeScript("user.json");
